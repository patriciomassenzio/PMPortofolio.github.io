Resumen del Equipo

Generador de perfil de equipo

Desrolle otro proyecto para probar y mejorar mi comprensión de JavaScript y Node.

Esta es una aplicación de línea de comandos de Node.js que toma información sobre los empleados en un equipo de ingeniería de software, luego genera una página web HTML que muestra resúmenes para cada persona, en un diseño completo y ordenado.

El proyecto se armó utilizando HTML, CSS, JavaScript y Node.js.
