Panel meteorológico

Un proyecto anterior que ayudó a solidificar mi comprensión de cómo utilizar las API web. En última instancia, esto se convertiría en un trampolín para el proyecto "¿A donde Vamos?"

Esta es una sencilla aplicación de panel meteorológico que solicita la entrada del usuario en forma de cualquier ciudad, después de lo cual se muestran los detalles del clima actual. Además, también se mostrarán los detalles del clima para los próximos 5 días.

Este proyecto fue realizado en Visual Studio Code con HTML, CSS, JavaScript con Fetch API.
