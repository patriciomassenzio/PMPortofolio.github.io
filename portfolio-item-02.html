A donde Vamos?

Familiarizarse con las API web

Una sencilla aplicación de viajes que solicita al usuario que busque en cualquier ciudad determinada sus respectivos detalles en el momento, como el clima, las vacantes en hoteles, los restaurantes y los eventos.

La idea vino después de pensar específicamente en la interconectividad del usuario a nivel local con cualquier ciudad determinada. Esta fue una oportunidad que me para demostrar mi conocimiento práctico de las API web, aplicadas de diferentes maneras para recuperar varios tipos de información local en cualquier ciudad ingresada.

Este proyecto se armó en gran medida en código de Visual Studio y github, utilizando HTML, CSS, JavaScript, API web y API RESTful.
