Generador de Contraseñas

"Seguro y confiable"

Al principio de mi educación, fue aquí donde comencé a comprender y apreciar completamente el desarrollo front-end y especialmente JavaScript.

Esta aplicación es un generador de contraseñas simple y seguro. Uno que podría implementarse en muchas aplicaciones. Comienza con una serie de indicaciones para el usuario, creadas para guiar a los usuarios a crear una contraseña específica única para ellos en función de ciertos parámetros para garantizar la seguridad.

Este proyecto fue realizado en Visual Studio Code con HTML, CSS y JavaScript.
