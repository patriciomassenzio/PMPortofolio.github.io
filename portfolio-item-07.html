Generador deREADME

Cree fácilmente archivos README.

Un proyecto sacado directamente de mi tiempo en la escuela, le tengo bastante cariño y aprecio ya que lo uso para crear todos mis archivos README para todos mis proyectos.

Cuando se le solicita, la terminal lleva al usuario a través de una serie de preguntas y entradas del usuario, y finalmente recopila la información en un archivo LÉAME cuidadosamente diseñado para cualquier proyecto determinado.

Este ingenioso proyecto se armó con node.js y npm.
